Netlify

Now we will cover how to post your website on to a site called Netlify which allows free web hosting up to a certain point. Starting out anything simple that you will be creating will not require any type of payment by you to host on this site but when you get further into it they will ask for hosting fees. The first step is putting your html and css files on github to give a repository to pull from in order to make the website. After this is done you will go to Netlify, create an account if you haven't already and then when you get to your home screen (shown below) you will click on "New site from Git". This will load your Github repositories and here you are able to load one of your projects on to the site. You can visit my website posted on Netlify if you'd like via this link.
